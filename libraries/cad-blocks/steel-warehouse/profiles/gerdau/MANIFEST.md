## Use Notes

- Treat these files as supplier CAD/BIM references, not verified structural
  design output.
- FreeCAD may need a DWG converter/importer configured before opening DWG
  files directly. If DWG import is unavailable, convert the DWG to DXF first.
- Prefer these files when a project specifies Gerdau W/HP, U, I industrial, or
  cantoneira profiles.
